About the Mandelbrot Set

The Mandelbrot set is the set of complex numbers c for which the sequence z₀ = 0, zₙ₊₁ = zₙ² + c does not diverge to infinity.

Constraint Theory Connection: This demonstrates how a single, simple geometric constraint (z² + c) can generate infinite complexity and self-similarity at all scales.

Smooth Coloring: We use continuous iteration counts to prevent color banding, calculated as: n + 1 - log(log(|z|)) / log(2)
