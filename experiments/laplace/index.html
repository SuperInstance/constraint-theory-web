About the Laplace Equation

The Laplace equation ∇²φ = 0 describes steady-state systems where the value at any point is the average of its neighbors. This creates a "smooth" field that satisfies boundary constraints everywhere simultaneously.

Physical Examples

  • Electrostatic potential in charge-free regions
  • Steady-state heat distribution
  • Fluid flow in porous media
  • Gravitational potential in empty space

Numerical Methods

  • Jacobi: Simple but slow (O(n) iterations)
  • Gauss-Seidel: Faster, uses updated values
  • SOR: Fastest with optimal ω (~1.8)

Using the Simulator

  • Select Boundary Mode and click edges to set fixed values
  • Use Source/Sink to add interior charges
  • Choose a Preset for classic configurations
  • Click Solve to iterate until convergence
  • Enable Animate to watch convergence in real-time
