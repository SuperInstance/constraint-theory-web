About Wave Interference

This simulator demonstrates how wave constraints combine through superposition:

  • Constructive Interference: Waves in phase add up (bright regions)
  • Destructive Interference: Waves out of phase cancel (dark regions)
  • Wave Equation: ψ = A·sin(kr - ωt + φ) / √r
  • Superposition: ψ_total = Σ ψ_i (sum of all wave contributions)

Geometric Interpretation: Each wave front is a constraint in spacetime. The interference pattern shows where constraints reinforce or negate each other.
