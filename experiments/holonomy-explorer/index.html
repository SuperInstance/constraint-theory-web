Holonomy Consistency Principle

Hol(γ) = I ⟺ Global Consistency

A constraint system is globally consistent if and only if holonomy around every cycle is identity. Non-identity holonomy reveals hidden inconsistencies.

⚠️ The Holonomy Test

Build a cycle of constraints. If you don't return to where you started, the constraint system is inconsistent. This is the fundamental test for solvability.

Understanding Holonomy

1

Build a Cycle

Compose rotations that return to the starting point geometrically.

2

Compute Holonomy

Multiply all rotation matrices. The result is the holonomy around the cycle.

3

Interpret Result

Identity = consistent. Non-identity = hidden inconsistency in constraints.

4

Fix or Accept

Either fix the constraint system or accept the inherent inconsistency (like Berry phase).
